# Adicionar Componente shadcn/ui

## Comando

// turbo

1. Adicionar componente (ex: Accordion):
```bash
npx shadcn@latest add accordion
```

## Componentes Disponíveis

Alguns componentes úteis ainda não instalados:
- `accordion` - Collapse/expand
- `alert` - Alertas coloridos
- `avatar` - Foto de perfil
- `calendar` - Seletor de data
- `checkbox` - Caixa de seleção
- `combobox` - Select com busca
- `command` - Command palette (⌘K)
- `date-picker` - Data picker completo
- `drawer` - Slide panel
- `menubar` - Menu horizontal
- `radio-group` - Radio buttons
- `slider` - Range slider
- `switch` - Toggle switch
- `tooltip` - Dicas hover

## Após Adicionar

1. Importar do `@/components/ui/[componente]`
2. Consultar docs: https://ui.shadcn.com/docs/components/[componente]
